# 甲状腺超声结构化数据方案 (Thyroid Ultrasound Structured Data Schema)

---

## 1. 概述

本方案定义了从甲状腺超声报告中提取结构化数据的字段及其规范。这些字段旨在捕获甲状腺检查的关键临床信息，以支持后续的数据分析、AI 辅助诊断和临床决策。

## 2. 核心字段

以下字段将作为甲状腺超声报告结构化数据的主要组成部分。所有字段均为可选，表示在报告中可能出现或不出现。

| 字段名 (英文)             | 字段名 (中文)      | 数据类型       | 描述                                                                                                                                                           | 示例                               |
| :------------------------ | :----------------- | :------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------- |
| `examDate`                | 检查日期           | String         | 进行超声检查的日期                                                                                                                                             | `2025-08-15`                       |
| `thyroidSizeLeft`         | 甲状腺左叶大小     | String         | 甲状腺左叶的尺寸（长*宽*厚）                                                                                                                                   | `4.5cm*1.5cm*1.2cm`                |
| `thyroidSizeRight`        | 甲状腺右叶大小     | String         | 甲状腺右叶的尺寸（长*宽*厚）                                                                                                                                   | `4.6cm*1.6cm*1.3cm`                |
| `isthmusThickness`        | 峡部厚度           | String         | 甲状腺峡部的厚度                                                                                                                                               | `0.3cm`                            |
| `thyroidEchogenicity`     | 甲状腺回声         | String         | 甲状腺实质的整体回声描述。<br>**建议值**: `均匀`, `不均匀`, `弥漫性增强`, `弥漫性减低`                                                                               | `回声尚均匀`                       |
| `nodules`                 | 甲状腺结节列表     | Array<Object>  | 甲状腺结节列表，每个对象描述一个结节的详细信息。如果无结节，则此数组为空。                                                                                     | `[见下方结节对象定义]`             |
| `lymphNodes`              | 颈部淋巴结         | String         | 颈部区域淋巴结的描述                                                                                                                                           | `双侧颈部未见肿大淋巴结`           |
| `diagnosisImpression`     | 诊断/印象          | String         | 医生对超声检查的诊断或初步印象                                                                                                                                 | `甲状腺多发结节（TI-RADS 4A类）`   |
| `recommendation`          | 建议               | String         | 医生根据检查结果提出的建议                                                                                                                                     | `建议穿刺活检`                     |

## 2.1 结节对象定义 (Nodule Object Definition)

当 `nodules` 字段中存在结节时，每个结节对象将包含以下字段：

| 字段名 (英文)     | 字段名 (中文)      | 数据类型 | 描述                                                                                                                                                           | 示例                               |
| :---------------- | :----------------- | :------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------- |
| `location`        | 结节位置           | String   | 结节在甲状腺内的具体位置。<br>**建议值**: `左叶上极`, `左叶中极`, `左叶下极`, `右叶上极`, `右叶中极`, `右叶下极`, `峡部`                                                 | `右叶中部`                         |
| `size`            | 结节大小           | Object   | 结节的尺寸（长*宽*高）。<br>`[见下方结节大小对象定义]`                                                                                                        | `[见下方结节大小对象定义]`         |
| `composition`     | 成分               | String   | 结节的内部成分。<br>**建议值**: `囊性或几乎完全囊性 (0分)`, `海绵状 (0分)`, `混合性囊实性 (1分)`, `实性或几乎完全实性 (2分)`                                       | `实性或几乎完全实性`               |
| `echogenicity`    | 结节回声           | String   | 结节的回声类型。<br>**建议值**: `无回声`, `低回声`, `等回声`, `高回声`, `极低回声`                                                                                 | `低回声`                           |
| `shape`           | 形态               | String   | 结节的形态，主要指高宽比。<br>**建议值**: `宽大于高 (0分)`, `高大于宽 (3分)`                                                                                     | `宽大于高`                         |
| `margin`          | 结节边界           | String   | 结节的边界描述。<br>**建议值**: `清晰`, `模糊`, `不规则`, `微小分叶`, `蟹足样`                                                                                     | `边界清晰`                         |
| `echogenicFoci`   | 回声灶             | String   | 结节内部的回声灶（钙化）类型。<br>**建议值**: `无或彗星尾征 (0分)`, `大粗钙化 (1分)`, `周边钙化 (2分)`, `点状强回声 (3分)`                                               | `点状强回声`                       |
| `bloodFlow`       | 结节血流 (CDFI)    | String   | 彩色多普勒血流成像显示的结节血流信号。<br>**建议值**: `无血流信号 (0级)`, `少量点状血流 (1级)`, `结节内部丰富血流 (2级)`, `结节周边丰富血流 (3级)`                               | `未见异常血流信号`                 |
| `tiRadsScore`     | TI-RADS 分级       | String   | 甲状腺影像报告和数据系统（TI-RADS）分级。<br>**建议值**: `1类`, `2类`, `3类`, `4A类`, `4B类`, `4C类`, `5类`, `6类`                                                | `4A类`                             |

## 2.2 结节大小对象定义 (Nodule Size Object Definition)

当 `size` 字段中存在结节大小信息时，该对象将包含以下字段：

| 字段名 (英文) | 字段名 (中文) | 数据类型 | 描述     | 示例   |
| :------------ | :------------ | :------- | :------- | :----- |
| `length`      | 长(mm)        | Number   | 结节的长度 | `5.0`  |
| `width`       | 宽(mm)        | Number   | 结节的宽度 | `4.0`  |
| `height`      | 高(mm)        | Number   | 结节的高度 | `3.0`  |

## 3. 扩展性与未来优化

-   **临床指南集成**: 将字段定义与最新的甲状腺结节诊疗指南（如 ATA, AACE 指南）对齐，提升数据规范性。
