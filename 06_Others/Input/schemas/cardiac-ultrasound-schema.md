# 心脏超声结构化数据方案 (Cardiac Ultrasound / Echocardiography Structured Data Schema)

---

## 1. 概述

本方案定义了从超声心动图报告中提取结构化数据的关键字段，涵盖心腔尺寸、心肌厚度、瓣膜功能、心脏功能评估及相关定量指标。字段设计兼顾二维超声、M 型、组织多普勒与应变成像等常见测量模式，便于后续进行心血管风险评估、趋势监测与临床决策支持。所有日期以 `YYYY-MM-DD` 表示；测量值遵循国际单位制（长度 mm、压力 mmHg、百分比 %）。

## 3. 数据校验与扩展建议

- **体表面积校正**: LVEDD、左心房内径等线性尺寸建议结合体表面积 (BSA) 进行校正；若报告未提供，可通过身高/体重派生并记录于 `reportNotes`。
- **计算一致性**: EF、FS、LVMI 等衍生指标应可由原始尺寸再计算；若系统直接导出结果，应保存原始测量方式和参考区间以便复核。
- **图像质量控制**: 当 `reportNotes` 指出图像质量欠佳或患者心律不齐时，应设置 `qualityFlag` 为 `需复测` 并提示补采样。
- **多模态融合**: 可扩展 `strainSegments`、`threeDVolumes`、`contrastUse` 等字段，以记录三维超声、对比增强超声或心脏 MRI 的互补信息。
- **时间序列管理**: 建议建立同一患者多次超声记录的版本链，实现趋势分析与疗效评估。
