# 眼底检查结构化数据方案 (Eye Fundus Examination Structured Data Schema)

---

## 1. 概述

本方案定义了从眼底检查报告中提取结构化数据的字段及其规范。这些字段旨在捕获眼底检查的关键临床信息，以支持后续的数据分析、挖掘和临床决策。

## 2. 核心字段

以下字段将作为眼底检查报告结构化数据的主要组成部分。所有字段均为可选，表示在报告中可能出现或不出现。

| 字段名 (英文)         | 字段名 (中文)      | 数据类型 | 描述                                     | 示例                 |
| :-------------------- | :----------------- | :------- | :--------------------------------------- | :------------------- |
| `examDate`            | 检查日期           | String   | 进行眼底检查的日期                       | `2025-08-15`         |
| `leftEye`             | 左眼检查           | Object   | 左眼眼底检查的详细信息                   | `[见下方眼部检查对象定义]` |
| `rightEye`            | 右眼检查           | Object   | 右眼眼底检查的详细信息                   | `[见下方眼部检查对象定义]` |
| `diagnosisImpression` | 诊断/印象          | String   | 医生对眼底检查的诊断或初步印象           | `糖尿病视网膜病变`   |
| `recommendation`      | 建议               | String   | 医生根据检查结果提出的建议               | `定期复查`           |

## 2.1 眼部检查对象定义 (Eye Exam Object Definition)

当 `leftEye` 或 `rightEye` 字段中存在眼部检查信息时，每个眼部检查对象将包含以下字段：

| 字段名 (英文)                | 字段名 (中文)        | 数据类型 | 描述                                                                                                                            | 示例                 |
| :--------------------------- | :------------------- | :------- | :------------------------------------------------------------------------------------------------------------------------------ | :------------------- |
| `opticDiscColor`             | 视盘颜色             | String   | 视盘的颜色描述。<br>**建议值**: `正常`, `淡红`, `红黄`, `苍白`, `颞侧苍白`                                                               | `红黄`               |
| `opticDiscMargin`            | 视盘边界             | String   | 视盘的边界描述。<br>**建议值**: `清`, `模糊`                                                                                     | `清`                 |
| `cupDiscRatio`               | 杯盘比               | Object   | 视盘杯盘比。<br>`[见下方测量值对象定义]`                                                                                                | `[见下方测量值对象定义]` |
| `opticDiscEdema`             | 视盘水肿             | String   | 视盘是否存在水肿。<br>**建议值**: `无`, `轻度`, `中度`, `重度`                                                                   | `无`                 |
| `opticDiscHemorrhage`        | 视盘出血             | String   | 视盘是否存在出血。<br>**建议值**: `无`, `有`                                                                                     | `无`                 |
| `macularEdema`               | 黄斑水肿             | Object   | 黄斑水肿的详细信息。<br>`[见下方黄斑水肿对象定义]`                                                                               | `[见下方黄斑水肿对象定义]` |
| `macularHemorrhage`          | 黄斑出血             | String   | 黄斑是否存在出血。<br>**建议值**: `无`, `有`                                                                                     | `无`                 |
| `macularDrusen`              | 黄斑玻璃膜疣         | String   | 黄斑是否存在玻璃膜疣。<br>**建议值**: `无`, `有`                                                                                 | `无`                 |
| `macularExudates`            | 黄斑渗出             | Object   | 黄斑渗出的详细信息。<br>`[见下方黄斑渗出对象定义]`                                                                               | `[见下方黄斑渗出对象定义]` |
| `retinalVesselAVRatio`       | 视网膜血管动脉/静脉比 | String   | 视网膜动脉与静脉的直径比                                                                                                        | `2:3`                |
| `retinalVesselNarrowing`     | 视网膜血管狭窄       | String   | 视网膜血管是否存在狭窄。<br>**建议值**: `无`, `轻度`, `中度`, `重度`                                                               | `无`                 |
| `retinalVesselSclerosis`     | 视网膜血管硬化       | String   | 视网膜血管是否存在硬化。<br>**建议值**: `无`, `轻度`, `中度`, `重度`                                                               | `无`                 |
| `retinalVesselHemorrhage`    | 视网膜血管出血       | Object   | 视网膜血管出血的详细信息。<br>`[见下方视网膜血管出血对象定义]`                                                                   | `[见下方视网膜血管出血对象定义]` |
| `retinalHemorrhage`          | 视网膜出血           | Object   | 视网膜出血的详细信息。<br>`[见下方视网膜出血对象定义]`                                                                           | `[见下方视网膜出血对象定义]` |
| `retinalExudates`            | 视网膜渗出           | Object   | 视网膜渗出的详细信息。<br>`[见下方视网膜渗出对象定义]`                                                                           | `[见下方视网膜渗出对象定义]` |
| `retinalCottonWoolSpots`     | 视网膜棉絮斑         | Object   | 视网膜棉絮斑的详细信息。<br>`[见下方视网膜棉絮斑对象定义]`                                                                       | `[见下方视网膜棉絮斑对象定义]` |
| `retinalPigmentaryChanges`   | 视网膜色素改变       | String   | 视网膜是否存在色素改变。<br>**建议值**: `无`, `有`                                                                               | `无`                 |
| `retinalDetachment`          | 视网膜脱离           | String   | 视网膜是否存在脱离。<br>**建议值**: `无`, `有`                                                                                   | `无`                 |
| `drSeverity`                 | 糖尿病视网膜病变严重程度 | String   | 糖尿病视网膜病变（DR）的严重程度分级。<br>**建议值**: `无DR`, `轻度非增殖性DR`, `中度非增殖性DR`, `重度非增殖性DR`, `增殖性DR` | `中度非增殖性DR`     |
| `dmeSeverity`                | 糖尿病黄斑水肿严重程度 | String   | 糖尿病黄斑水肿（DME）的严重程度分级。<br>**建议值**: `无DME`, `轻度DME`, `中度DME`, `重度DME`                                     | `轻度DME`            |

## 2.2 黄斑水肿对象定义 (Macular Edema Object Definition)

当 `macularEdema` 字段中存在黄斑水肿信息时，该对象将包含以下字段：

| 字段名 (英文) | 字段名 (中文) | 数据类型 | 描述                                     | 示例   |
| :------------ | :------------ | :------- | :--------------------------------------- | :----- |
| `present`     | 是否存在      | String   | 是否存在黄斑水肿。<br>**建议值**: `无`, `有` | `有`   |
| `centralInvolvement` | 累及中心凹    | String   | 黄斑水肿是否累及中心凹。<br>**建议值**: `否`, `是` | `是`   |
| `hardExudates` | 硬性渗出      | String   | 黄斑区是否存在硬性渗出。<br>**建议值**: `否`, `是` | `是`   |
