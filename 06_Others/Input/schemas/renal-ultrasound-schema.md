# 肾脏超声结构化数据方案 (Renal Ultrasound Structured Data Schema)

---

## 1. 概述

本方案定义了从肾脏超声检查报告中提取结构化数据的字段及其规范。这些字段旨在捕获肾脏超声检查的关键临床信息，以支持后续的数据分析、挖掘和临床决策。

## 2. 核心字段

以下字段将作为肾脏超声检查报告结构化数据的主要组成部分。所有字段均为可选，表示在报告中可能出现或不出现。

| 字段名 (英文)         | 字段名 (中文)      | 数据类型 | 描述                                     | 示例                 |
| :-------------------- | :----------------- | :------- | :--------------------------------------- | :------------------- |
| `examDate`            | 检查日期           | String   | 进行肾脏超声检查的日期                   | `2025-08-15`         |
| `kidneySide`          | 肾脏侧别           | String   | 检查的肾脏。<br>**建议值**: `左肾`, `右肾`, `双肾` | `双肾`               |
| `leftKidney`          | 左肾               | Object   | 左肾的详细检查信息。<br>`[见下方肾脏对象定义]` | `[见下方肾脏对象定义]` |
| `rightKidney`         | 右肾               | Object   | 右肾的详细检查信息。<br>`[见下方肾脏对象定义]` | `[见下方肾脏对象定义]` |
| `leftUreter`          | 左输尿管           | Object   | 左输尿管的详细检查信息。<br>`[见下方输尿管对象定义]` | `[见下方输尿管对象定义]` |
| `rightUreter`         | 右输尿管           | Object   | 右输尿管的详细检查信息。<br>`[见下方输尿管对象定义]` | `[见下方输尿管对象定义]` |
| `bladder`             | 膀胱               | Object   | 膀胱的详细检查信息。<br>`[见下方膀胱对象定义]` | `[见下方膀胱对象定义]` |
| `diagnosisImpression` | 诊断/印象          | String   | 医生对肾脏超声检查的诊断或初步印象       | `双肾多发囊肿`       |
| `recommendation`      | 建议               | String   | 医生根据检查结果提出的建议               | `定期复查`           |

## 2.1 肾脏对象定义 (Kidney Object Definition)

当 `leftKidney` 或 `rightKidney` 字段中存在肾脏信息时，每个肾脏对象将包含以下字段：

| 字段名 (英文)          | 字段名 (中文)      | 数据类型      | 描述                                                                                                                                                           | 示例                 |
| :--------------------- | :----------------- | :------------ | :------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------- |
| `length`               | 长度               | Object        | 肾脏的最大长度。<br>`[见下方测量值对象定义]`                                                                                                                  | `[见下方测量值对象定义]` |
| `width`                | 宽度               | Object        | 肾脏的最大宽度。<br>`[见下方测量值对象定义]`                                                                                                                  | `[见下方测量值对象定义]` |
| `thickness`            | 厚度               | Object        | 肾脏的最大厚度。<br>`[见下方测量值对象定义]`                                                                                                                  | `[见下方测量值对象定义]` |
| `cortexThickness`      | 皮质厚度           | Object        | 肾皮质的厚度。<br>`[见下方测量值对象定义]`                                                                                                                    | `[见下方测量值对象定义]` |
| `echogenicity`         | 回声               | String        | 肾实质的回声情况。<br>**建议值**: `正常`, `增强`, `减低`                                                                                                      | `正常`               |
| `cysts`                | 囊肿列表           | Array<Object> | 肾脏囊肿列表，每个对象描述一个囊肿的详细信息。<br>`[见下方囊肿对象定义]`                                                                                     | `[见下方囊肿对象定义]` |
| `stones`               | 结石列表           | Array<Object> | 肾脏结石列表，每个对象描述一个结石的详细信息。<br>`[见下方结石对象定义]`                                                                                     | `[见下方结石对象定义]` |
| `hydronephrosis`       | 肾盂积水           | Object        | 肾盂积水的详细信息。<br>`[见下方肾盂积水对象定义]`                                                                                                            | `[见下方肾盂积水对象定义]` |
| `mass`                 | 占位               | Object        | 肾脏占位的详细信息。<br>`[见下方肾脏占位对象定义]`                                                                                                            | `[见下方肾脏占位对象定义]` |
| `perirenalFluid`       | 肾周积液           | String        | 肾周围是否存在积液。<br>**建议值**: `无`, `有`                                                                                                                | `无`                 |

## 3. 扩展性与未来优化

-   **临床指南集成**: 将字段定义与最新的肾脏超声检查临床指南进行更紧密的集成，确保数据采集的规范性和完整性。
