🏥 健康顾问系统

基于知识库的智能健康咨询平台

🚀 快速启动指南

按照以下步骤启动健康顾问系统：

1. 安装依赖

pip install fastapi uvicorn pydantic

2. 启动API服务

cd docs/Yilan python api_server_example.py

3. 测试系统

# 运行集成测试 python health_advisor_integration.py # 或访问API文档 open http://localhost:8000/docs
