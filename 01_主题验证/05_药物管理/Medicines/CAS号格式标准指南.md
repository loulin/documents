# CAS号格式标准指南

## CAS号格式规范

### 标准格式
- **正确格式**: CA + 数字 + 连字符 + 数字
- **示例**: CA657-24-9, CA11061-68-0

### 处理原则

#### 1. 复合制剂CAS号处理
- **原则**: 选择主要活性成分的CAS号
- **示例**: 
  - 二甲双胍/西格列汀 → 使用二甲双胍CAS号
  - 胰岛素复合制剂 → 使用胰岛素CAS号

#### 2. 前缀标准化
- **统一前缀**: 所有CAS号添加"CA"前缀
- **格式清理**: 移除特殊字符，保留数字和连字符

#### 3. 验证规则
- **格式验证**: ^CA[\d\-]+$
- **长度验证**: 通常10-15字符
- **字符验证**: 只包含CA、数字、连字符
